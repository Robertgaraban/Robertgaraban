```
+------------------------------------------------------------------+
|                    AI AGENT ARCHITECTURE                         |
+-------------------------+----------------------------------------+
|  AGENT.md               |  Context, rules, and agent persona     |
|  (per system)           |  scoped to each domain                 |
+-------------------------+----------------------------------------+
|  SKILL.md               |  Defined deliverables + completion     |
|  (per task)             |  criteria for each task                |
+-------------------------+----------------------------------------+
|  Multi-model            |  Claude (Anthropic) + OpenAI GPT       |
|                         |  selected by task type and domain      |
+-------------------------+----------------------------------------+

+-------------------------+----------------------------------------+

+-------------------------+----------------------------------------+
```

## ⚓ Maritime Domain Glossary

```
STCW    — Standards of Training, Certification and Watchkeeping (IMO)
DGMM    — Dirección General de la Marina Mercante (Spain's Maritime Authority)
DPC     — Diretoria de Portos e Costas (Brazil's Maritime Authority)
COLREGs — International Regulations for Preventing Collisions at Sea
USV     — Unmanned Surface Vehicle
```
